# Glyph 🔣

Interactive algorithmic ASCII art generator + artwork playground.

Now supports:
- **Artwork presets** (Glyph's exported default config is the canonical baseline)
- **Live playground controls** per artwork
- **Adrian variants** (`Submit this version`) with comment + config snapshot
- **Notes for Glyph** (feedback queue Glyph can review before creating new art)
- **API ingestion** so Glyph can auto-publish new works

Deploy target: **Vercel**.

## Algorithms

| Algorithm | Description |
|-----------|-------------|
| **Matrix Rain** | Cascading columns of digital glyphs |
| **Fluid Smoke** | Rising smoke wisps with fluid dynamics |
| **Reaction-Diffusion** | Gray-Scott morphogenesis — organic patterns |
| **Fire** | Classic demoscene fire effect |
| **Wave Interference** | Circular waves from multiple point sources |
| **Cellular Automata** | 1D rules cascading into complex structures |
| **Particle Swirl** | Particles orbiting gravitational attractors |

## Controls

- **Algorithm** — pick your simulation
- **Palette** — 9 colour schemes (matrix, cyan, amber, fire, ice, etc.)
- **Character Set** — ASCII ramps from minimal to dense
- **Font Size** — affects grid resolution (smaller = more detail)
- **Speed** — simulation speed multiplier
- **Algorithm-specific params** — each algo has unique tweakable settings
- **Effects** — scanlines, vignette, glow post-processing
- **Seed** — reproducible randomness. Same seed = same art.

### Keyboard Shortcuts

| Key | Action |
|-----|--------|
| `Space` | Pause/Play |
| `R` | Restart |
| `S` | Random seed |

## How It Works

Everything is algorithmically rendered. Numpy-style typed arrays, physics simulations, cellular automata — pure code. The aesthetic is ASCII characters on dark backgrounds with CRT-style post-processing.

No AI image generation. No external APIs. Just maths.

## Data model (Supabase)

Run this SQL once:

```sql
create table if not exists glyph_comments (
  id bigint generated by default as identity primary key,
  artwork_id text,
  author text default 'adrian',
  comment text not null,
  created_at timestamptz default now()
);

create table if not exists glyph_variants (
  id bigint generated by default as identity primary key,
  artwork_id text not null,
  author text default 'adrian',
  comment text,
  config jsonb not null,
  created_at timestamptz default now()
);

create table if not exists glyph_artworks (
  id text primary key,
  payload jsonb not null,
  created_at timestamptz default now()
);
```

Set env vars in Vercel project:
- `SUPABASE_URL`
- `SUPABASE_SERVICE_ROLE_KEY`
- `GLYPH_DEPLOY_TOKEN`

## Glyph auto-publish API

`POST /api/artworks` with header `Authorization: Bearer $GLYPH_DEPLOY_TOKEN` and JSON payload:

```json
{
  "id": "unique-artwork-id",
  "title": "Artwork title",
  "description": "Optional",
  "presentation": { "mode": "animated" },
  "defaultConfig": {
    "algo": "matrix",
    "palette": "matrix",
    "ramp": "digital",
    "fontSize": 14,
    "speed": 1,
    "seed": 42,
    "fx": { "scanlines": true, "vignette": true, "glow": false },
    "params": {}
  }
}
```
